## Key Responsibilities and Achievements

- **Training and Development System**: Engineered and deployed a comprehensive training and development platform (Web & Mobile), utilizing Laravel and REST API for robust backend architecture and Flutter for a cross-platform mobile user experience.
- **Internal Job Portal**: Developed and maintained an internal job portal system using Laravel, streamlining recruitment workflows and improving organizational hiring efficiency.
- **Work Order System**: Built an efficient work order management system leveraging Next.js for a responsive frontend and PostgreSQL for reliable, scalable database management.
- **HRIS System (SobatHR)**: Architected and launched a full-stack HRIS solution (Web Admin & Mobile App), featuring a Laravel backend, Next.js web interface, and Flutter mobile application. Successfully deployed the mobile application on both the App Store and Google Play Store.
- **AI Integration**: Collaborated with advanced AI tools to engineer fully functional, highly secure, user-friendly, and scalable software systems, optimizing development workflows and ensuring robust performance.
